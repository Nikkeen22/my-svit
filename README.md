# 📺 My Svit — Kids Video Platform

Спеціалізована веб-платформа (аналог YouTube), розроблена для безпечного перегляду дитячих мультфільмів та пізнавального контенту.

---

### 🌟 Ключові особливості
* **🎞 Стрімінг контенту:** Оптимізований інтерфейс для перегляду відео без зайвого відволікання.
* **👼 Дитячий режим:** Простий та інтуїтивно зрозумілий дизайн, з яким впорається навіть дитина.
* **📱 Responsive Design:** Сайт адаптований під планшети та смартфони, щоб мультики можна було дивитися будь-де.
* **⚡ Швидкість:** Побудовано на **React**, що забезпечує миттєве перемикання між відео без перезавантаження сторінки.

---

### 🛠 Технологічний стек
* **Frontend:** React.js (Component-based architecture)
* **Styling:** CSS3 / Modern Flexbox & Grid
* **Infrastructure:** Vercel для хостингу та швидкого доступу до контенту.
* **Data Handling:** JavaScript (ES6+) для керування станом плеєра та списком відтворення.
